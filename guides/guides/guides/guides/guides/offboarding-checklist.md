# Offboarding Checklist

## Purpose
This checklist is used to ensure secure and complete removal of user access when an employee leaves the organization.

## Checklist

### 1. Confirm departure details
- Full name
- Last working day
- Department
- Manager approval

### 2. Disable access
- Block sign-in to Microsoft 365
- Reset password if required
- Revoke active sessions

### 3. Remove license
- Remove or reassign Microsoft 365 license where appropriate

### 4. Secure company data
- Review OneDrive ownership
- Review mailbox retention needs
- Transfer business-critical data if required

### 5. Remove access to shared resources
- Shared mailboxes
- Distribution groups
- Security groups
- Shared drives
- Business applications

### 6. Recover company equipment
- Laptop
- Charger
- Mobile phone
- Access cards
- Any other assigned equipment

### 7. Document completion
- Record date of offboarding
- Record administrator responsible
- Confirm all tasks completed

## Outcome
The offboarding process is complete once all access is revoked and company assets are recovered.
