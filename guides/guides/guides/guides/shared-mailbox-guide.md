# Shared Mailbox Guide

## Purpose
This guide explains how shared mailboxes are used and how access should be managed.

## What is a shared mailbox?
A shared mailbox is a mailbox that allows multiple users to read and send emails from a common address, such as:
- info@company.com
- support@company.com
- accounts@company.com

## When to use a shared mailbox
- Departmental communication
- Customer support inboxes
- Finance or admin communication

## Administration Process

### 1. Create shared mailbox
- Open Microsoft 365 Admin Center
- Go to Teams and Groups or Exchange Admin Center
- Create shared mailbox
- Assign mailbox name and email address

### 2. Grant access
- Add users who need access
- Assign Read and Manage permissions
- Assign Send As permissions if required

### 3. Test access
- Confirm mailbox appears in Outlook
- Confirm users can read mail
- Confirm users can send from the mailbox if approved

## Best Practices
- Only grant access to authorized users
- Review access regularly
- Remove access when users change roles or leave

## Risks
Shared mailboxes should be monitored carefully to avoid unauthorized access and misuse.
