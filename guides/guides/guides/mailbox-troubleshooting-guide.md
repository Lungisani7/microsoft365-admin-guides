# Mailbox Troubleshooting Guide

## Purpose
This guide outlines common troubleshooting steps for Microsoft 365 mailbox and Outlook-related issues.

## Common Issues
- User cannot log in to email
- Outlook not syncing
- Mailbox full
- Emails not sending or receiving
- Shared mailbox not appearing

## Troubleshooting Steps

### 1. Confirm account status
- Check if the user account is active
- Check if the Microsoft 365 license is assigned
- Confirm mailbox exists

### 2. Check credentials
- Confirm correct email address is being used
- Reset password if needed

### 3. Check internet connectivity
- Confirm the device has internet access
- Test web access to Outlook

### 4. Test Outlook Web Access
- Ask the user to log in through Outlook on the web
- If web access works, issue may be local to the Outlook app

### 5. Check Outlook client
- Restart Outlook
- Recreate Outlook profile if required
- Remove and re-add account
- Check for updates

### 6. Mail flow checks
- Confirm mailbox is not full
- Check spam or junk folder
- Confirm external email delivery is functioning

### 7. Escalation
If the issue persists:
- collect screenshots
- record error messages
- escalate to senior administrator or Microsoft support if needed

## Outcome
The mailbox issue should be resolved or escalated with enough detail for further investigation.
