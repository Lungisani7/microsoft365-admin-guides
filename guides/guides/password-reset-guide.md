# Password Reset Guide

## Purpose
This guide explains the standard process for resetting a user's password in a Microsoft 365 environment.

## Steps

### 1. Verify identity
Before resetting a password, confirm the identity of the user through approved methods such as:
- employee ID
- manager confirmation
- registered contact details

### 2. Open admin portal
- Sign in to Microsoft 365 Admin Center
- Navigate to Users
- Select Active Users

### 3. Locate user
- Search for the user by name or email address
- Open the user profile

### 4. Reset password
- Select Reset Password
- Generate a temporary password or create one manually
- Ensure "Require this user to change password when they first sign in" is selected

### 5. Communicate securely
- Send the temporary password through an approved secure channel
- Do not send passwords through insecure methods

### 6. Confirm access restored
- Ask the user to log in
- Confirm access to email and required systems

## Notes
Password reset requests should always be logged according to company support procedures.
