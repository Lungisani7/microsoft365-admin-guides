# User Onboarding Checklist

## Purpose
This checklist is used to ensure that new employees are properly provisioned with the accounts, access, and tools they need to begin work.

## Checklist

### 1. Confirm user details
- Full name
- Job title
- Department
- Manager
- Start date
- Office location
- Contact number

### 2. Create user account
- Create Microsoft 365 user account
- Assign company email address
- Set temporary password
- Force password change at first login

### 3. Assign license
- Assign the appropriate Microsoft 365 license
- Confirm services included with the license

### 4. Configure access
- Add user to required distribution groups
- Add user to security groups
- Grant access to shared folders
- Grant access to required applications

### 5. Configure mailbox
- Confirm mailbox creation
- Test mail flow
- Add shared mailbox access if required

### 6. Device preparation
- Prepare laptop or desktop
- Install required applications
- Configure antivirus and updates
- Join device to company environment if applicable

### 7. Communication
- Send welcome email
- Send login details securely
- Provide first-day IT support contacts

## Completion
The onboarding process is complete once all required systems, accounts, and access permissions have been verified.
