# Microsoft 365 Admin Guides

## Overview
This repository contains practical Microsoft 365 administration guides and support documentation for common user and IT support tasks.

## Purpose
The purpose of this repository is to demonstrate knowledge of Microsoft 365 administration, user support processes, and IT operational documentation.

## Included Guides
- User onboarding checklist
- Password reset guide
- Mailbox troubleshooting guide
- Shared mailbox guide
- Offboarding checklist

## Included Templates
- New user request template
- Access review template

## Why This Project Matters
Microsoft 365 administration is a key part of many IT support, systems support, and technical administration roles. This repository demonstrates practical process knowledge and documentation skills relevant to real business environments.
